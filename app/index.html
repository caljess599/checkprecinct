HOW TO USE THIS FORM
For the street name, DO NOT include:
• Street type (RD, AVE, LN, etc.)
• Street direction (W, E, NW, SE), whether it appears before or after the street name
• Unit number

Examples:
• W Main St becomes Main
• Numbered street names are written as so: 1st, 2nd, 3rd, 4th, 5th, etc.
• "Half" streets are written like so: 9 1/2

The search will return all relevant matches.

How does this work?
This script makes API calls to public GIS data available at Open Data Charlottesville
Source code available on Github.
